HTML

HTML adalah Hypertext Markup Languange.

HTML merupakan bahasa mark up yang mudah untuk dipahami oleh pengguna.

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup.

CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.
